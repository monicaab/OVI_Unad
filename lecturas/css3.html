Lectura No. 3: ¿Qué es CSS3?

CSS3:Hoja de estilos en cascada(Cascade Style Sheet)

Este no es un lenguaje de programación, css es un lenguaje que nos permite resumir, controlar el aspecto y la presentación de los elementos html que tiene un sitio web.
Es decir CSS nos permite dar propiedades modificando el aspecto de nuestros elementos en la página web, en este caso css3 es una actualización de CSS la cual in fluye nuevas cosas, pero no modifica cosas anteriores. Por ejemplo en esta versión se han mejorado las opciones de sombreado y redondeado, hasta funciones avanzadas de movimiento y transformación. Sus funciones más destacadas son:

  • animaciones
  • rotación de elementos
  • distintos usos de fuentes
  • Es 100% gratuito

PRINCIPALES USOS DE CSS3

Animaciones y transiciones en CSS: uno de los mejores y mayores logros de css3 es que ofrece la posibilidad de añadir animaciones que se verán alteradas gracias a fotogramas y transiciones, es decir que puede cambiar la apariencia y comportamiento de un elemento, por ejemplo cuando el cursor pasa sobre dicho elemento

Layout con columnas: en este caso el número de columnas suele ser mayor que la mayoría de páginas web esto gracias a la propiedad “Multicolumn Layout” de css3 el cual te permite especificar el número de columnas así como el ancho de ellas.

Gradientes: este te permite cambiar el color de tus elementos en la página sino que también te permite crear gradientres y así se mejorar el tiempo de respuesta del sitio creado.

Rotación de elementos: en este caso la propiedad “tranform” te permite cambiar el tamaño de los elementos y así también moverlos, este se puede aplicar a imágenes, botones y cualquier tipo de elementos de tipo 2D, esto también se puede realizar para elementos tipo 3D pero es obvio que los resultados varían.

Uso de fuentes: este nos permite agregar fuentes desde librerías de como google front, ya que cuenta con gran número de fuentes gratuitas que se pueden añadir a css sin mayor inconveniente, sin embargo de ellas también se puede cambiar las fuentes y especificar el tamaño.
