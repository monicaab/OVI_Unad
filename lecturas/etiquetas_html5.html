Lectura No. 3: Etiquetas HTML5

Definición

Son la base de Html5 como lenguaje de programación, es decir es aquello que vemos entre corchetes en el código de nuestra web.
Las etiquetas se clasifican segun su proposito en:

  • Fragmentos de Texto
  • Orientados a aspectos informaticos
  • Orientados a medicion

Tomado de: https://es.wikipedia.org/wiki/Archivo:HTML5_logo_black.svg#/media/Archivo:HTML5_logo_and_wordmark.svg

FRAGMENTOS DE TEXTO

ETIQUETAS DESCRIPCION
<strong> Fragmento de texto importante o palabras clave.
<em> Fragmento de texto enfatizado respecto a la frase que lo contiene. Ejemplo: xxxxxxxx
<mark> Fragmento de texto resaltado, simulando estar marcado con rotulador amarillo. Ejemplo: XXXXXX
<i> Fragmento de texto con voz o tono alternativo al resto.
(Se debe tener en cuenta que en versiones anteriores de HTML la etiqueta simbolizaba un fragmento de texto en cursiva.)
<b> Fragmento de texto sin importancia destacable fines utilitarios. Ejemplo: (xxxxxxxx)
(Se debe tener en cuenta que en versiones anteriores de HTML la etiqueta simbolizaba un fragmento de texto en negrita.)
<u> Fragmento de texto para nombres propios o escritura incorrecta intencionada (sic).
<s> Fragmento de texto inexacto o que ya no es relevante, es decir un error o palabra inexacta. Ejemplo: (xxxxxxx)
<span> Fragmento de texto sin significado (útil para seleccionar).
<cite> Fragmento de texto con el título de un trabajo creativo: obras, libros, etc.

Estas etiquetas nos ayudaran a elegir un fragmento de texto y darles un significado especial.

Con estas etiquetas de HTML, podemos modificar el significado de la información que contien en:

ETIQUETAS ATRIBUTOS DESCRIPCION
<sup> Se utiliza para indicar Superíndice (242).
<sub> Se utiliza para indicar Subíndice (242).
<small> Anotaciones menores pequeñas puntualizaciones.
<q> <cite> Cita o frase extraída de otro contexto, pero no sirve como un hipervínculo.
<dfn> <title> Definición (término que posteriormente será definido).
<abbr> <title> Abreviatura o acrónimo.

ETIQUETAS DESCRIPCION
<br> Salto de línea (crear una nueva línea, es decir simula un enter).
<wbr> Oportunidad de salto de línea (división silábica con guion, para evitar que el texto desborde el contenido).

Los siguientes son etiquetas para saltos de línea: estas nos ayudan para comenzar u texto en una línea nueva.

ORIENTADA A ASPECTOS INFORMATICOS

Estas etiquetas de HTML nos ayudan representar variables matemáticas o informáticas.

ETIQUETAS ATRIBUTOS DESCRIPCION
<kbd> Muestra una entrada de información del usuario (generalmente, combinación de teclado).
<samp> Muestra la salida de información de un programa informático.
<small> Anotaciones menores pequeñas puntualizaciones.
<var> Permite indicar variables utilizadas en (contexto matemático o informático).
<time> <Datetime (formato orientado a maquinas)> Indica una fecha / hora legible para humanos, con formato para máquinas opcional.
<data> <Value(texto equivalente orientado a maquinas)> Información equivalente orientada a máquinas.
<code> Fragmento de código fuente (en línea).

ETIQUETAS DE MEDICION

Estas etiquetas nos sirven cuando ya hemos publicado una página pero queremos modificar o añadir información pero queremos reflejar que esos cambios han sido modificados o insertados después.

ETIQUETAS ATRIBUTOS DESCRIPCION
<ins> <cite, Datetime(nos permite añadir una URL con información adicional)> Fragmento de texto o información añadida a posteriori de la publicación inicial.
<del> <cite, Datetime(nos permite añadir fecha y hora de cuando se eliminó la información)> Fragmento de texto o información eliminada a posteriori de la publicación inicial, no se elimina del todo solo que se tacha con un estilo visual que sugiera que ya no es válida.
