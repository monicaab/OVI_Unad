Lectura No. 2: ¿Qué es HTML5?

HTML5:Lenguaje de Marcas de Hipertexto(Hypertext Markup Language)

es un lenguaje de programación el cual es el encargado de definir el formato que tiene que llevar el contenido de un documento, así que nos ayuda en la elaboración de páginas web como un estándar, pues define una estructura básica y un código para la definición del contenido de una página web como texto, imágenes, videos, etc.

Tomado de: https://es.wikipedia.org/wiki/Archivo:HTML5_logo_black.svg#/media/Archivo:HTML5_logo_and_wordmark.svg

Estructura

Toda página web hecha con Html5, por complicada que sea o parezca, se divide en dos partes fundamentales para dar un orden:

  • Cabecera (head)
  • Cuerpo (body)
Adicionalmente todas las páginas comienzan y terminan con una etiqueta:
  • comienza: <html>
  • termina: </html>
Es decir que toda página web cuenta con su encabezado y su cuerpo.

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>

<body>
Content of the document......
</body>

</html>

TIPO DE DOCUMENTO

por medio de esta línea, podemos indicar al navegador que el contenido del archivo que estamos creando es código Html y tambien podemos indicar qué tipo de Html concretamente, pues hay varias formas de escribirlo en función de la versión de Html que estemos usando al escribir el código.

<!DOCTYPE html>

IDIOMA 

<html lang="es">

HTML5 Esta versión nos obliga a definir qué lenguaje o idioma estamos usando para los contenidos. Pues se debe indicar en la etiqueta esta definición que abre el código aunque no se pueden indicar varios idiomas en una misma página.

TITULO

Existen una serie de elementos que no se ven directamente en el contenido de las páginas y el (title) o título definido en la cabecera es uno de estos.
Una parte muy importante son las etiquetas (title )con las que se define el título de cada una de las páginas web.
No de la web completa sino de cada página del sitio web. Ya que se debe tener una etiqueta para indicar que empieza el título y otra para indicar que el título ha terminado.

<title>Title of the document</title>

META DESCRIPCIÓN 

<html lang="es">

Las metaetiquetas, etiquetas meta o elementos meta (también conocidas por su nombre en inglés, metatags o meta tags) son etiquetas HTML que se incorporan en el encabezado de una página web y que resultan invisibles para un visitante normal, pero de gran utilidad para navegadores u otros programas que puedan valerse de esta información. Su propósito es el de incluir información (metadatos) de referencia sobre la página: autor, título, fecha, palabras clave, descripción, juego de caracteres, etc..

CABECERA

El elemento HTML head provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos..

<head></head>

CUERPO 

<html lang="es">

De las dos partes en que se divide un documento HTML (HEAD y BODY), BODY es la segunda. BODY es la etiqueta usada para indicar el cuerpo de un documento HTML, es la parte donde se describe el contenido de la página (su estructura, su forma, sus colores, texto, y todo lo visual).
